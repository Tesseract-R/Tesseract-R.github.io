Seminar No.1730 Tensor Complementarity Problem

Title: Tensor Complementarity Problem
Reporter: Prof. Liqun Qi (Department of Mathematics The Hong Kong Polytechnic University)
Time: 2018-12-13 (Thursday) 15:00
Place: G507

Abstract: In the last three years, the tensor complementarity problem has been studied extensively, from theory to solution methods and applications. In this talk, we survey these developments. This is a joint work with Zhenghai Huang.
